# Git Workflow (3 Areas)

```
Workspace → Staging Area → Local Repository
```

### 1. Workspace
Your normal project folder.

### 2. Staging Area
Files you prepare for commit:
```bash
git add .
```

### 3. Local Repository
Stored commits:
```bash
git commit -m "message"
```

Check tracking status:
```bash
git status
```
