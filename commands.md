# Common Git Commands

## Initialize Git
```bash
git init
```

## Track files
```bash
git add filename
git add .
```

## Commit files
```bash
git commit -m "message"
```

## Check status
```bash
git status
```

## Check logs
```bash
git log
git log --oneline
```

## See differences
```bash
git diff
```

## Remove cached file
```bash
git rm --cached filename
```
