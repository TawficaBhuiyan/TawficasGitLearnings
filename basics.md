# Git & GitHub Basics

## ⭐ What is Version Control?
Version control helps you keep track of:
- Different versions of your project  
- What changed and when  
- Collaboration between developers  

Example:
- Day1 (v1)
- Day2 (v2)
- Day3 (v3)

## ⭐ Why Git?
- 70% developers use Git  
- Free, open-source  
- Fast, distributed version control  

## ⭐ Git vs GitHub

| Git | GitHub |
|-----|--------|
| Tool/Software | Service |
| Local version control | Cloud hosting |
| CLI based | GUI based |
| Maintained locally | Maintained online |

## ⭐ Who created Git?
- Created by **Linus Torvalds**
- Initial release: **April 7, 2005**
- GitHub founded: **2008**, owned by Microsoft
